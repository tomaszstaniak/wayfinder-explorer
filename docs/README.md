# Wayfinder docs

This folder separates durable project state from working notes.

- `state/` is tracked source-of-truth documentation: what exists now, current architecture, and accepted product direction.
- `plans/`, `explorations/`, `archive/`, and `superpowers/` are local working notes and are ignored by Git.

Keep `state/` small. If a note is speculative, stale, or implementation-only, it belongs outside `state/`.
